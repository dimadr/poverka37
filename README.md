# poverka37.ru

Статический сайт компании ООО «Поверка».

## Цели проекта

* Предоставить клиентам информацию об услугах компании.
* Показать контакты и способы связи.
* Обеспечить минимальную стоимость сопровождения сайта.

## Архитектурные решения

### Статический HTML

Сайт состоит из статических файлов и не использует CMS.

Причины:

* отсутствие базы данных;
* отсутствие серверной логики;
* минимальная поверхность атаки;
* отсутствие необходимости в регулярных обновлениях CMS и плагинов;
* простое резервное копирование и перенос на любой хостинг.

### Отказ от WordPress

WordPress был признан избыточным для задач сайта.

Функциональность сайта ограничивается предоставлением информации о компании и контактных данных, поэтому использование CMS не имеет практических преимуществ.

### Геофильтрация

Доступ к сайту ограничен по географическому признаку.

Компания оказывает услуги только на территории Российской Федерации, поэтому внешний трафик не представляет коммерческой ценности и создаёт дополнительную нагрузку за счёт ботов, сканеров и автоматизированных систем.

### Карта

На странице отображается изображение карты.

Интерактивная карта загружается только по запросу пользователя.

Это снижает объём загружаемых данных и уменьшает количество внешних запросов при открытии сайта.

## Сопровождение

Для публикации изменений достаточно заменить файлы на веб-сервере.

Дополнительных сервисов, баз данных и процедур миграции не требуется.

Основной принцип проекта — минимально необходимая сложность.
